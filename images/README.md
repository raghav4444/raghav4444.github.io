# Project Images

**IMPORTANT:** Add your project screenshots here for best performance!

Place your project screenshot images in this folder with the following naming convention:

- `studex.png` or `studex.jpg` - For Studex project
- `studify.png` or `studify.jpg` - For Studify project  
- `omnifood.png` or `omnifood.jpg` - For Omnifood project

## How to Add Images:

1. Take screenshots of your live projects:
   - Studex: https://raghav4444.github.io/Studex/
   - Studify: https://studifyka.netlify.app/
   - Omnifood: https://omnifood-raghvendra.netlify.app/

2. Save them with the names above in this folder

3. The portfolio will automatically use these local images once you add them.

## Recommended Image Specifications:
- Width: 1200px or larger
- Format: PNG or JPG
- Aspect Ratio: 16:9 or similar
- File Size: Optimized (under 500KB if possible)

**Note:** Currently, the portfolio uses screenshot API URLs. For better performance and reliability, add your actual screenshots here!
